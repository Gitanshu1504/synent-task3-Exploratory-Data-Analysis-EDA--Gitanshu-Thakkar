Project Title

Exploratory Data Analysis (EDA) on Netflix Dataset

Problem Statement:

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Netflix dataset to identify trends, patterns, and relationships in the data. The analysis helps understand Netflix's content distribution, production trends, and genre popularity.

Dataset Details:

Dataset Name: Netflix Dataset
File Format: CSV
Main Features:
Show ID
Type
Title
Director
Cast
Country
Date Added
Release Year
Rating
Duration
Genre (listed_in)
Description

Approach:

Loaded the dataset using Pandas.
Examined the dataset structure and summary statistics.
Checked for missing values.
Converted the date_added column into datetime format.
Performed correlation analysis on numeric features.
Identified yearly content addition trends.
Visualized Movies vs TV Shows distribution.
Identified the top 10 content-producing countries.
Identified the top 10 most popular genres.
Derived business insights from the visualizations.
Results:

Movies are more prevalent than TV Shows on Netflix.
The number of titles added to Netflix increased significantly over time.
The United States is the leading producer of Netflix content.
International Movies, Dramas, and Comedies are among the most common genres.
Correlation among the available numeric features is relatively low.
